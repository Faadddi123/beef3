En tant que concepteur, vous concevrez des maquettes **MOBILE **chaque page du site web : la page d'accueil, la page du catalogue de produits et la page d'inscription. Ces maquettes doivent saisir l'essence de FashionHub et garantir une expérience utilisateur optimale.
​

En tant que développeur, vous concevrez la maquette de la page d'accueil et la transformerez en pages web fonctionnelles en utilisant HTML, CSS et JavaScript pour mettre en avant l'excitation de FashionHub. Vous intégrerez des fonctionnalités telles qu'une section pour les clients satisfaits, leurs témoignages inspirants et un appel à l'action pour s'inscrire. Vous devrez également ajouter une section FAQ interactive en utilisant JavaScript, permettant aux utilisateurs de consulter les questions fréquemment posées et d'agrandir les réponses d'un simple clic. Cette page doit s'adapter parfaitement à tous les écrans.
​

En tant que développeur, vous concevrez la maquette de la page du catalogue de produits en utilisant HTML, CSS et JavaScript pour présenter les différents produits de mode FashionHub. Détails des produits, catégories et marques . Assurez-vous que cette page est conviviale sur les appareils mobiles.
​

En tant que développeur, vous concevrez la maquette de la page d'inscription en utilisant HTML, CSS et JavaScript. Les visiteurs pourront s'inscrire et saisir leurs informations pour devenir membres de FashionHub. Vous devrez également vérifier que les informations saisies par l'utilisateur sont valides (en utilisant JavaScript REGEX). Cette page doit fonctionner parfaitement sur tous les écrans.
​

En tant que développeur, vous adaptrez le menu de navigation pour la version mobile du site web. Implémentez un menu burger (une icône avec trois barres horizontales) qui, lorsqu'elle est cliquée, affichera les options de navigation de manière conviviale. Utilisez JavaScript pour animer cette fonctionnalité et garantir une navigation fluide sur les appareils mobiles. Assurez-vous que le menu est intuitif et facilement accessible pour les utilisateurs mobiles.
